## 🚀 QUICK REFERENCE

### Start Development

```bash
cd aura-digitals-web
npm install
npm run dev
# Open http://localhost:5173
```

### Build for Production

```bash
npm run build
# Creates dist/ folder
```

### Key Files to Edit

| Task                   | File                                  |
| ---------------------- | ------------------------------------- |
| Change text/headings   | `src/utils/constants.js`              |
| Change colors          | `tailwind.config.js`                  |
| Change WhatsApp number | `.env`                                |
| Change logo/brand name | `src/utils/constants.js` (APP_NAME)   |
| Add new section        | Create in `src/components/organisms/` |
| Add new page           | Create in `src/pages/`                |
